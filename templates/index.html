The Flow

From Sensor to Screen in Seconds

Five stages, one seamless pipeline — your machine health is always visible.

01
ESP32 Collects Sensor Data
Temperature, vibration, and current sensors continuously read machine health every 2 seconds from your industrial equipment.
02
MQTT Transmission to HiveMQ Cloud
Data is published as JSON payloads over TLS-encrypted MQTT to the HiveMQ Cloud broker — reliable and secure.
03
Python Subscriber Cleans & Features
Our data cleaner validates ranges, removes outliers, and engineers rolling mean/std features for the ML model.
04
Random Forest Predicts Fault
The trained model outputs a fault probability — Normal (<35%), Warning (35–65%), or Fault (>65%).
05
Dashboard Updates in Real-Time
Flask SocketIO pushes the result to your browser every 2 seconds — gauges, charts, and status banners all live.

Sensor Coverage

Every Parameter that Matters

Our sensor array captures the most critical indicators of machine health continuously and accurately.

🌡️
Temperature
DHT11 sensor monitors operating temperature — elevated readings indicate friction, overload, or cooling failure.
Range: 0°C – 150°C
📳
Vibration
ADXL345 accelerometer detects abnormal vibration patterns — early warning for bearing wear, imbalance, or misalignment.
Range: 0g – 10g RMS
⚡
Current
ACS712 hall-effect sensor measures motor current draw — spikes signal mechanical resistance or electrical faults.
Range: 0A – 20A
💧
Humidity
Ambient humidity monitoring protects against condensation and corrosion — critical for long-term machine reliability.
Range: 0% – 100% RH
